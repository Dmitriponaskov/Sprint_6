Тесты для сервиса «Яндекс.Самокат»
Этот проект содержит автоматизированные UI-тесты для учебного сервиса «Яндекс.Самокат».
Тесты написаны на Python с использованием Selenium, pytest и Allure.
Применяется паттерн Page Object Model (POM) для удобства поддержки кода.

Что тестируется
1. Вопросы и ответы (FAQ)
Проверяется, что при нажатии на любой из 8 вопросов в разделе «Вопросы о важном» открывается правильный ответ.
2. Оформление заказа самоката
Проверяется полный сценарий заказа:
через кнопку «Заказать» в шапке сайта
через кнопку «Заказать» внизу страницы
Используются два разных набора данных (Иван Петров и Анна Сидорова).
После оформления проверяется:
появление окна с подтверждением «Заказ оформлен»
возможность перейти на главную страницу, нажав на логотип «Самоката»
корректность ссылки на Яндекс и успешный редирект на Дзен ()
Структура проекта
data/ — содержит тестовые данные и URL-адреса
pages/ — классы страниц (MainPage, OrderPage) и базовый класс BasePage
tests/ — тестовые сценарии:
test_faq_accordion.py — тесты для FAQ
test_order_flow.py — тесты для оформления заказа
conftest.py — настройка фикстур (браузер, куки и т.д.)
requirements.txt — список зависимостей
README.md — этот файл
Как запустить
Установите Python 3.8 или новее.
Установите браузер Mozilla Firefox.
Скачайте и установите geckodriver, добавьте его в PATH.

Установите зависимости: 

pytest tests/ --alluredir=./allure-results

Чтобы посмотреть отчёт в браузере: 

allure serve ./allure-results
